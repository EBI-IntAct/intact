Project description
  • IntAct provides a freely available, open source database system and analysis tools for protein interaction data. All interactions are derived from literature curation or direct user submissions and are freely available.

Members of the IntAct consortium:
  • EBI - European Bioinformatics Institute, Cambridge, United Kingdom
  • MPI-MG - Max-Planck-Institut fuer Molekulare Genetik, Berlin, Germany
  • SIB - Swiss Institute of Bioinformatics, Geneva, Switzerland
  • SDU - Odense University, Denmark
  • UB1 - University of Bordeaux, France
  • CNB-CSIC - Centro National de Biotecnologia, Madrid, Spain
  • HUJI - The Hebrew University of Jerusalem, Israel
  • GSK - GlaxoSmithKline plc, Harlow, United Kingdom
  • MINT - University Tor Vergata, Rome, Italy

Software:
    IntAct is an open source project hosted on SourceForge. You can access the IntAct group page, the IntAct home page, or browse the SVN repository.

    Additional software developed within the IntAct project:

    • ProViz graph visualisation system.
    • Targets application to predict targets for pull-down experiments
    • MiNe application computes minimal connecting network for protein sets.
      Example: Q12277 and Q08491.
Data:

Availability:
  • All IntAct data and software is freely available to all users, academic or commercial, under the terms of the Apache License, Version 2.0.
Status:
  • Source code release 0.8 on April 1st, 2004.
  • Monthly XML data release since February 1st, 2004.
  • Production since summer 2003.
Direct data submission:
  • IntAct encourages and welcomes direct user submissions of protein interaction data to be published in peer-reviewed publications. Please contact datasubs@ebi.ac.uk for curator assistance.
