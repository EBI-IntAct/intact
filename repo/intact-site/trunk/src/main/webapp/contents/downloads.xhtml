Downloads

  • IntAct is released weekly. All IntAct data and software is freely available to all users, academic or commercial, under the terms of the Apache License, Version 2.0, from #{ftp}
  • Contents of the IntAct FTP directory:
    • all.zip: entire contents of this directory, compressed.
    • cv: controlled vocabularies.
    • psi1: IntAct data in PSI MI 1.0 format. See the PSI website for a detailed format description. This format is now superseeded by the PSI 2.5 format.
    • psi25: IntAct data in PSI MI 2.5 format. See the PSI website for a detailed format description.
    • various: Data subsets for exchange with collaborating databases.
