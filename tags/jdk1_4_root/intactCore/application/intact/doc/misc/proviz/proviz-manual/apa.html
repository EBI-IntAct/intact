Requirements

Before installing ProViz, ascertain that your system fulfills the following requirements:

  • Tulip v1.2.5

  • Qt v3.0 or above

  • OpenGL (shipped with XFree 4.x)

  • glut

  • Optionnal: the Mozilla web browser, or at least a browser runnable with the command "mozilla" (for example a symlink to "konqueror")

Compiling from the sources

In order to compile and install ProViz properly, you need:

  • the GNU C++ compiler, versions 2.95, 2.96, 3.2 or higher. Versions ranging from 3.0 to 3.2 excluded may be able to compile although we don't support them it since they are not provided as stable.

  • The make utility.

  • Qt development version 3.0 or higher.

  • X11 development version (include files are often missing)

  • OpenGL and glut development version

As ProViz's sources use the GNU automake/autoconf system, the compilation process goes through the classical configure/make/make install process. Important options are:

  • --prefix: where to install proviz bin/ lib/ share/... directories
  • --with-qt-{includes, libraries}: where make should find Qt headers and libraries, respectively
  • --with-tulip-config: the directory containing the tulip-config script, if it is not in the PATH environment variable
