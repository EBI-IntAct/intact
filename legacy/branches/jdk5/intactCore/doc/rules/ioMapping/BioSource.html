Mapping of External data to BioSource objects


This document describes how external data is mapped into the BioSource object. The normal data source is NEWT, http://www.ebi.ac.uk/newt.
IntAct attribute in BioSource
 Newt data element
shortLabel Swiss-Prot OS code
fullName Scientific name
taxId taxid
new xref:
Database = TaxId, primaryIdentifier = taxid, secondaryIdentifier = Swiss-Prot OS code 		-
