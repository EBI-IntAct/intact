term: 3d-r-factors: free R-Factor and working R-Factor for the quality of the crystallographic model
definition: R-factors are measures of the quality of a crystallographic model. During refinement of a crystal structure a small test set of refinement data may be set aside and used to assess the quality of the refinement process. The R-factor that is calculated using such a test set of data is the free R-factor whilst the R-factor calculated for the data that is used in refinement is the working R-factor. If no test set was used in refinement, the single, overall R-factor is given.
example: working 24.4%, free 28.4%
