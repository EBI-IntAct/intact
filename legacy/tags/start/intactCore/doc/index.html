IntactCore

The first development phase for IntAct. It should provide a fully functional system.

Current components

  1. Implementation overview
  2. Javadoc documentation of data model, including UML class diagram

Software used

We are currently using the following packages which are not part of Java JDKs:
  • Castor (http://www.castor.org) for reading/writing Java objects to/from XML. Castor is distributed as part of the IntAct package under a BSD-like license.
  • OJB (http://jakarta.apache.org/ojb/) for object-relational mapping. OJB is distributed as part of the IntAct package under the Apache license.
  • Struts (http://jakarta.apache.org/struts/index.html. Struts is distributed as part of the IntAct package under the Apache license.
  • Display (http://edhill.its.uiowa.edu/display/. The Display library is distributed as part of the IntAct package under the Artistic License.
  • Poseidon (http://www.gentleware.com) for UML modeling.
